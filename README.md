# Travel Behaviour Analysis

 Overview
This coursework project analyses travel behaviour patterns using large-scale mobility datasets. The study focuses on stay-at-home trends, trip frequency, and travel distance to understand how people move over time.



Objectives
- Analyse mobility behaviour patterns
- Examine trip frequency and travel distance
- Identify trends using data visualisation
- Model relationships using regression techniques
- Compare serial and parallel data processing performance

 Tools & Technologies
- Python
- Pandas (serial processing)
- Dask (parallel processing)
- Matplotlib / Seaborn (visualisation)
- Scikit-learn (regression models)



 Key Features
- Data cleaning and preprocessing
- Visualisation of mobility trends
- Regression modelling (linear & polynomial)
- Performance comparison (Pandas vs Dask)


Key Findings
- Most travel occurs over short to medium distances
- Moderate trip frequency is more common than high-frequency travel
- Travel frequency decreases as distance increases
- The relationship between distance and frequency is non-linear
- Parallel processing improves computational efficiency
